# Fake News Detector

Projet de détection automatique des fake news en anglais avec TF-IDF et Logistic Regression.

## Structure
- `data/` : dataset CSV
- `models/` : modèle entraîné
- `src/` : code Python
- `notebooks/` : exploration des données
- `app.py` : WebApp Streamlit
- `main.py` : CLI test

## Installation
```bash
git clone <repo>
cd fake_news_detector
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
